# OOPS (Object-Oriented Programming) Examples

This folder contains small Python examples that demonstrate core OOP concepts and patterns.

Files included

- `animal.py` - Simple base class example showing class definition and instance usage.
- `duck_typing.py` - Demonstrates duck typing: different classes with the same method interface.
- `dunder_method.py` - Examples of Python "dunder" (magic) methods like `__str__`, `__repr__`, and operators.
- `hierarchical_inheritance.py` - Shows hierarchical inheritance where multiple subclasses inherit from a single base class.
- `method_overriding.py` - Demonstrates how subclasses override parent class methods and call `super()`.
- `multilevel_inheritance.py` - Example of multilevel inheritance (A -> B -> C) and method resolution.
- `multiple_inheritance.py` - Shows multiple inheritance and how Python resolves attribute lookup (MRO).
- `single_inheritance.py` - Basic single-inheritance example with parent and child classes.

How to use

Run any example directly with Python. For example:

```
python single_inheritance.py
```
