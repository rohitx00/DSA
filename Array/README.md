# Array Folder

This folder contains Java programs related to basic array operations in Data Structures and Algorithms (DSA).

## Files

### BasicArray.java

A simple Java program that demonstrates basic array operations. It prompts the user to input 5 integer elements into an array, stores them, and then displays the array elements.

### ReverseArray.java

This program takes user input for an array of 5 integers, displays the original array, and then reverses the array using a recursive swap method. It utilizes the `swap` function from `SwapArrayEle.java` to perform the reversal.

### SwapArrayEle.java

A program that illustrates swapping elements in an array. It takes 5 integer inputs, displays the original array, swaps the first and last elements (indices 0 and 4), and prints the modified array. It also contains a static `swap` method that can be used by other classes for swapping array elements.
