# Linear Search Folder

This folder contains implementations of linear search algorithms in Java.

## Files

- **LinearSearchArray.java**: Implements linear search on an integer array to find the index of a target element. It takes user input for array elements and searches for a hardcoded target (19).

- **LinearSearchArrayInRange.java**: Implements linear search on an integer array within a specified range (start to end indices) to find the index of a target element. It searches between indices 1 and 3 in the input array.

- **LinearSearchString.java**: Implements linear search on a string to check if a specific character is present. It takes user input for a name and a character to search, converting both to lowercase for case-insensitive search.

- **LinearSearchStringInRange.java**: Implements linear search on a string within a specified range to check if a specific character is present. It searches between indices 1 and 3 (inclusive) in the input string.

- **MaxEleInArray.java**: Finds the maximum value in an integer array by iterating through all elements and comparing them.

- **MinEleInArray.java**: Finds the minimum value in an integer array by iterating through all elements and comparing them.

- **EvenDigitCoutInArray.java**: Counts the number of elements in an integer array that have an even number of digits. It takes user input for array elements and uses a helper method to determine if a number has an even digit count.

- **SearchIn2DArray.java**: Implements linear search on a 2D integer array to check if a target element is present. It iterates through all rows and columns to find the target value.
